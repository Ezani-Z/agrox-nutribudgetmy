# AI Tools Disclosure

## PutraHack 2026 Compliance Statement

In accordance with PutraHack 2026 guidelines, we hereby disclose all Artificial Intelligence (AI) tools used in the development of **NutriBudget MY**.

---

## Tools Used

| Tool | Provider | Purpose | Usage Context |
|------|----------|---------|---------------|
| **Claude Code** | Anthropic | PRD writing, technical architecture, code assistance | Used for creating the Product Requirements Document, designing the constraint-satisfaction algorithm, and generating React component templates |
| **Lovable** | Lovable Labs | Frontend UI/UX development, prototyping | Primary low-code platform for building the web application interface |
| **GitHub Copilot** | GitHub | Code completion, boilerplate generation | Optional assistance for repetitive coding tasks (if used during development) |

---

## Declaration

### What AI Assisted With:
- Document drafting and structure (PRD, README)
- Technical architecture design and documentation
- Code templates and component scaffolding
- Algorithm logic formulation
- Risk assessment and feature prioritization

### What Was Human-Directed:
- Core problem identification and ideation
- Strategic decisions on feature scope and priorities
- Final validation of technical approach
- Business logic and domain knowledge (Malaysian school canteen context)
- All AI outputs were reviewed, edited, and approved by human team members

---

## Transparency Commitment

We commit to:
1. **Full disclosure** of all AI tools used in development
2. **Human oversight** on all AI-generated outputs
3. **Original ideation** - The core concept and problem-solving approach were human-developed
4. **Validation** - All technical specifications and calculations were verified by the team

---

## Compliance

This disclosure meets the requirements set forth by PutraHack 2026 for AI tool transparency. Teams are permitted to use AI tools provided they are disclosed, and the final submission represents original work with appropriate attribution.
